## 📋 Overview

A modern and responsive travel agency website built with *HTML5* and *Tailwind CSS*.  
NextTrip helps travelers discover destinations, explore famous landmarks, and learn about travel services through a clean and modern interface.

⚙️ Installation

Prerequisites

· A modern web browser (Chrome, Firefox, Safari, Edge)
· Git (optional, for cloning)

Local Setup

1. Clone the repository
   ```bash
   git clone https://github.com/filopater23106-cloud/NextTrip-Tourism-Website.git
   ```
2. Navigate to project folder
   ```bash
   cd NextTrip-Tourism-Website
   ```
3. Open the project
   · Option A: Double-click index.html to open directly in browser
   · Option B: Use Live Server in VS Code for auto-refresh
   ```bash
   # If you have VS Code with Live Server extension
   right-click index.html → Open with Live Server
   ```

Note: No build tools or dependencies required – the website runs directly in any browser!

🎨 Customization Guide

🖌 Colors

The website uses a consistent dark theme color palette:

Element Tailwind Class Description
Background bg-slate-900 Dark blue-gray background
Accent text-amber-400 Golden amber for highlights
Primary Text text-white White text
Secondary Text text-slate-300 Light gray text
Borders border-slate-700 Subtle border color

🖼 Images

To customize images:

1. Replace existing images in the Images/ folder
2. Recommended image sizes:
   Section Recommended Size
   Hero background 1200×800px
   Landmark cards 400×300px
   Logo 48×48px

📝 Content

To modify website content:

1. Open index.html in any text editor
2. Update the following sections:
   · Text content – Headlines, descriptions, and buttons
   · Statistics – Numbers and labels in the stats section
   · Services – Service descriptions and icons
   · Destinations – Add or remove landmark cards
   · Navigation – Update menu items if needed
